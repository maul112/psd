Knime Tutuorial#

Install KNIME dan Python Script#

Kali ini saya akan memberikan cara menggunakan software KNIME sederhana. yang pertama kita harus download software KNIME di url https://www.knime.com/downloads lalu install. Tampilan software KNIME awal-awal akan seperti berikut:

Kali ini saya ingin menggunakan Python Script. Awal-awal kalian tidak akan menemukan Python Script pada KNIME dibagian kategori Scripting, adanya cuman Java. Maka dari itu kita install terlebih dahulu Python Script nya caranya adalah:

  1. Klik menu di pojok kanan atas, lalu pilih Install Extention.

  1. Lalu cari Python (disini tidak ada karena saya sudah install).

pilih yang namanya sesuai sama berikut:

Jika sudah install, KNIME akan meminta untuk restart software. Setelah itu jika sudah muncul node Python berarti proses instalasi berhasil.

Menarik data dari PostgreSQL ke KNIME#

  1. Drag node “PostgreSQL Connector” kedalam Workflow.

  1. Lalu klik 2 kali pada node tersebut untuk konfigurasi. Sesuaikan data-data dengan konfigurasi PostgreSQL yang kalian miliki.

  1. Lalu kita butuh konfigurasi driver PosstgreSQL di KNIME. Klik Preference pada halaman utama KNIME. Lalu pilih databases lalu klik add.

Pilih file driver PostgreSQL dengan cara klik add file. Setelah menambahkan driver klik Find Driver Classes lalu klik Ok (download driver di url https://jdbc.postgresql.org/download/).

  1. Ketika ingin memilih tabel, gunakan node “DB Table Selector” lalu tuliskan schema dan nama tabel.

  1. Setelah itu, sambungkan antar node seperti digambar (drag kotak pink kecil samping node PostgreSQL Connector dan sambungkan ke kotak pink di node DB Table Selector)

  1. Untuk membaca database kita perlu menambahkan node “DB Reader”.

  1. Lalu run semua node hingga menunjukkan lingkaran hijau kecil dibawah node.

  1. Lalu kita tambahkan node Python Script (Legacy).

  1. Lalu konfigurasi Python dengan cara pergi ke Preferences → KNIME → Python (Konfigurasi juga di Python (Legacy)). Pilih manual lalu pilih path python yang anda gunakan.

  1. Lalu run node Python. Pada gambar terlihat sudah keluar untuk tampilan tabel yang sudah kita tarik dari PostgreSQL ke KNIME
